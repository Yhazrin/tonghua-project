# Iteration 1: 安全与基础设施收口

**目标**: 建立可承载生产数据的安全底座，确保认证、审计与敏感信息保护到位。

## 任务清单

- [ ] **统一认证链路**
  - [ ] 实现 Access/Refresh Token 双令牌机制
  - [ ] 实现 Logout 与 Token Revoke (黑名单) 策略
- [ ] **角色与权限控制 (RBAC/ABAC)**
  - [ ] 完善监护人、审核员、超级管理员角色
  - [ ] 为管理端接口补充资源级权限判断
- [ ] **儿童信息专项保护**
  - [ ] 实现敏感字段 (如家长联系方式、孩子真名) 的字段级加密
  - [ ] 实现前端展示的字段级脱敏工具
- [ ] **审计日志系统**
  - [ ] 建立统一的审计记录拦截器/工具
  - [ ] 重点审计：状态变更 (审核、支付、权限修改)
- [ ] **基础设施加固**
  - [ ] 重新激活或优化 HMAC 请求签名中间件 (考虑多端适配)
  - [ ] 完善限流 (Rate Limiting) 策略
